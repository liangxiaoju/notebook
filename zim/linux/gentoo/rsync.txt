====== rsync ======
Created 星期五 02 三月 2012

TAGS: rsync

rsync - a fast, versatile, remote (and local) file-copying tool


=== server: ===
	* 配置/etc/rsyncd.conf
	reference:
		pid file = /var/run/rsyncd.pid
		use chroot = yes
		read only = yes

		[test]
			path = /tmp
			comment = For Test
			max connections = 0
			read only = no
			write only = no

	* start daemon
		rsync --daemon
		or
		/etc/init.d/rsyncd start

=== client: ===
	rsync [opt] ''rsync://ip_addr/test test''
	option:
		* -a archive mode
		* -r recurse
		* -d transfer directories without recursing
		* -l 仅拷贝链接本身
		* -L 拷贝链接指向的内容
		* -p 保留权限
        * -u 只拷贝新的文件
        * -P 断点续传
